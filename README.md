# NetManager

**NetManager** — это приложение для управления сетевыми ресурсами или настройками, разработанное на Python. Проект представляет собой графическое приложение (судя по наличию файла `interface.py`) и, вероятно, предназначен для упрощения администрирования сети.

## 📋 Описание проекта

Проект, судя по структуре, нацелен на создание исполняемого приложения с графическим интерфейсом. Он включает в себя:
*   Модуль для сборки приложения.
*   Основную логику и интерфейс.
*   Скомпилированные файлы для запуска.

## 🚀 Основной функционал

Точный функционал не указан, но на основе структуры можно предположить следующие возможности:
*   Взаимодействие с пользователем через графический интерфейс.
*   Выполнение основных сетевых задач (например, проверка подключения, сканирование портов, управление конфигурациями).
*   Сборка в автономное приложение для удобства использования.

## 📁 Структура проекта

```
NetManager/
├── .idea/                  # Файлы конфигурации IDE (PyCharm)
├── build/                  # Вспомогательные файлы для сборки
│   └── NetManager/         # Промежуточные файлы сборки
├── dist/                   # Готовые исполняемые файлы приложения
├── NetManager.spec         # Файл спецификации для PyInstaller
├── interface.py            # Модуль с графическим интерфейсом пользователя (GUI)
└── main.py                 # Главный исполняемый файл (точка входа)
```

## 🔧 Установка и запуск

### Способ 1: Запуск из исходного кода

1.  **Клонирование репозитория**:
    ```bash
    git clone https://github.com/xDefox/NetManager.git
    cd NetManager
    ```

2.  **Установка зависимостей**:
    Убедитесь, что у вас установлен Python 3. Зависимости могут быть перечислены в файле `requirements.txt` (которого нет в репозитории). Если он отсутствует, установите необходимые библиотеки вручную (например, `tkinter` для GUI, `PyInstaller` для сборки).
    ```bash
    pip install -r requirements.txt # или pip install pyinstaller
    ```

3.  **Запуск приложения**:
    ```bash
    python main.py
    ```

### Способ 2: Запуск готового приложения

В папке `dist` находится собранная версия приложения. Вы можете запустить её напрямую (обычно это исполняемый файл `NetManager.exe` для Windows), без установки Python.

## 📄 Использование

После запуска откроется графический интерфейс, созданный с помощью `interface.py`. Взаимодействуйте с приложением через кнопки, поля ввода и другие элементы управления для выполнения сетевых задач.
